## Free Tier

**Goal:** Hook users, demonstrate value, drive upgrades

### Included Features

| Category | Feature |
|----------|---------|
| **Core Workflow** | Target → Aim → Shot workflow |
| **Targets** | Thesis documentation, confidence level, abort triggers |
| **Aims** | Playable + Monitor aim types (manual only) |
| **Shots** | Position tracking, stop loss/exit trigger fields |
| **Dashboard** | Basic dashboard, P&L display |
| **Scoring** | 2-3 metrics only (Win Rate + Basic PPD) |
| **Referrals** | Get personal referral code |

### Limitations (Upgrade Drivers)

| Feature | Limitation |
|---------|------------|
| **Scoring timing** | After trade closes only (no live grading) |
| **Scoring granularity** | Aggregate metrics only (no per-shot breakdown) |
| **Monitor Aims** | Manual creation only, no "what if" P&L |
| **Analytics** | Basic win/loss count |
| **AI Features** | None |
| **Benchmarks** | None |

## Premium Plus Tier

**Goal:** High-value upsell for serious traders

### Included Features

Everything in Premium, plus:

| Category | Feature |
|----------|---------|
| **AI Target Coach** | Full lifecycle coaching |
| **"Why/Why Not?" Prompts** | Reflective thesis hygiene prompts |
| **Causal Analysis** | Deeper thesis pattern analysis |
| **Risk Gap Detection** | AI suggests risks you may have missed |
| **Historical Patterns** | "Similar theses in 2021 had X% success" |
| **Strength/Weakness ID** | AI identifies your trading patterns |
| **Pattern Warnings** | Real-time alerts based on your history |
| **Weekly AI Summary** | Email digest of insights |
| **Thinking Quality Coach** | Long-term coaching on decision quality |
| **Alternative Strategies** | Options, DCA, covered call suggestions |
| **Priority Support** | Faster response times |

### AI Target Coach Components

1. **Theory Analysis (Pre-Trade)**
   - Risk assessment score
   - Likelihood rating for projections
   - Counter-arguments / bear case prompts
   - Similar historical patterns
   - Related opportunities (correlated tickers)
   - Correlation data between suggested tickers

2. **Active Coaching (During Trade)**
   - Catalyst reminders ("Earnings in 3 days")
   - Target price alerts
   - Thesis validation ("This pullback is within your bounds")
   - Exit trigger reminders

3. **Post-Trade Review**
   - Outcome vs thesis analysis (skill vs luck)
   - Missed signal identification
   - Theory improvement suggestions
   - Pattern recognition across trades

4. **Post-Exit Guidance**
   - Remaining upside alerts
   - Re-entry opportunity signals
   - "Took profits appropriately" vs "exited prematurely" analysis
   - FOMO prevention

### AI Pattern Analysis

| Analysis Type | Example Insight |
|---------------|-----------------|
| Strength ID | "You excel at tech sector momentum plays — 78% accuracy" |
| Weakness ID | "Your timing on earnings plays is poor — wait for post-earnings" |
| Calibration | "Your 8+ confidence calls underperform your 5-6 calls" |
| Discipline | "The 10% you ignore stop losses cost 40% of your losses" |
| Patterns | "You exit winners early and hold losers too long" |
| Sector | "Avoid healthcare — 35% accuracy there" |
| Timing | "Monday entries outperform Friday entries" |

## Referral Program

### Referral Codes

- Every user gets a unique referral code at signup
- Auto-generated (e.g., `MATT7X2K`)
- Can customize once (vanity code: `MATTTRADES`)
- Shareable link: `outvestments.com/r/CODE`

### Rewards Structure

| Event | Referrer Gets | Referee Gets |
|-------|---------------|--------------|
| Signup | — | 14-day Premium trial |
| Converts to Premium | 1 free month | 10% off first month |
| Stays 3+ months | Bonus month | — |

### Ambassador Program (Future)

Power referrers get enhanced benefits:
- Higher referral rewards
- Early access to features
- Public recognition

## Promo Code System

### Code Types

| Type | Effect | Example |
|------|--------|---------|
| `tier_grant` | Grants tier access for X days | "PREMIUMWEEK" = 7 days Premium |
| `trial_extension` | Extends trial period | "EXTRA7" = +7 days |
| `percent_off` | % discount on subscription | "LAUNCH20" = 20% off |
| `percent_off_limited` | % off for N months only | "TRY50" = 50% off 3 months |
| `fixed_off` | $ discount | "5OFF" = $5 off first month |
| `free_months` | X months free | "PODCAST3" = 3 months free |
| `upgrade_discount` | Discount on tier upgrade only | "UPGRADE30" = 30% off |

### Code Constraints

- `valid_from` / `valid_until` — Time-limited
- `max_uses` — Scarcity
- `max_uses_per_user` — Prevent abuse
- `new_users_only` — Acquisition codes
- `min_tier` — Upgrade codes only

## Technical Requirements

### Database Tables

1. User tier fields (extend `users`)
2. `user_acquisition` — UTM/referral/affiliate tracking
3. `referrals` — Referral program
4. `promo_codes` — Promotion system
5. `promo_redemptions` — Code usage tracking
6. `affiliates` — Affiliate partners
7. `affiliate_conversions` — Affiliate tracking
8. `global_config` — Global overrides
9. `tier_features` — Feature flags per tier

### Integration Points

- Stripe for subscriptions
- Feature flag system in codebase
- Usage metering for AI features (cost control)
- Email service for trial expiration, upsells
